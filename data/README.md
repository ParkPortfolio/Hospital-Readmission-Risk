# Data

This folder contains the public dataset used for the Hospital Readmission Risk Dashboard.

The dataset was obtained from an online public source and was used for portfolio/educational analysis. No private or personally identifiable patient information is included.

## Files

- `diabetic_data.csv` — raw public dataset used for the analysis.
- ## Source

Dataset source: (https://archive.ics.uci.edu/dataset/296/diabetes+130-us+hospitals+for+years+1999-2008)]
